## English

Pick random problems with advanced filtering options on BOJ (solved.ac) and LeetCode.

### BOJ (solved.ac)

| Feature | Description |
|---------|-------------|
| **Tier Range** | Set desired difficulty from Bronze V to Ruby I |
| **Class Filter** | Filter problems by Class 1–10 |
| **Algorithm Tag Chips** | Selected tags shown as chips, individually removable with x |
| **Solved Filter** | All / Exclude my solved / Only my solved (auto-detects handle) |
| **Exclude Obscure** | Exclude problems with ≤N solvers (customizable, default 100) |
| **Quick Fetch** | Double-click to fetch immediately |

### LeetCode

| Feature | Description |
|---------|-------------|
| **Difficulty Checkboxes** | Select multiple: Easy, Medium, Hard |
| **Tag Chips** | Select/remove tags as chips (Array, DP, Graph, etc.) |
| **Accepted Filter** | Exclude obscure problems by min accepted count (default ≥1000) |
| **Solved Filter** | All / Exclude my solved / Only my solved (LeetCode session) |
| **Count** | Set number of problems to pick (1–20) |

> Random is not available for Programmers and SWEA.

## 한국어

백준(solved.ac)과 LeetCode에서 다양한 필터를 적용하여 랜덤으로 문제를 뽑을 수 있습니다.

### 백준 (solved.ac)

| 기능 | 설명 |
|------|------|
| **티어 범위** | Bronze V ~ Ruby I 까지 원하는 난이도 범위 지정 |
| **클래스 필터** | Class 1~10 기준으로 문제 필터링 |
| **알고리즘 태그 칩** | 선택한 태그가 칩으로 표시, x 클릭으로 개별 제거 |
| **풀이 필터** | 전체 / 내가 푼 문제 제외 / 내가 푼 문제에서만 (핸들 자동 감지) |
| **듣보 문제 제외** | 맞은 사람 N명 이하인 문제 제외 (기본 100명) |
| **즉시 가져오기** | 뽑힌 문제를 더블클릭으로 바로 가져오기 |

### LeetCode

| 기능 | 설명 |
|------|------|
| **난이도 체크박스** | Easy, Medium, Hard 복수 선택 |
| **태그 칩** | Array, DP, Graph 등 태그를 칩으로 선택/제거 |
| **정답자 수 필터** | 정답자 N명 이상인 문제만 (기본 1000명) |
| **풀이 필터** | 전체 / 내가 푼 문제 제외 / 내가 푼 문제에서만 |
| **개수 설정** | 뽑을 문제 수 지정 (1~20개) |

> 프로그래머스와 SWEA는 랜덤 기능을 지원하지 않습니다.
