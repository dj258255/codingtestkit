## English

Pick random problems with advanced filtering options on LeetCode, Codeforces, Programmers, and SWEA.

### LeetCode

| Feature | Description |
|---------|-------------|
| **Difficulty Checkboxes** | Select multiple: Easy, Medium, Hard |
| **Tag Chips** | Select/remove tags as chips (Array, DP, Graph, etc.) |
| **Accepted Filter** | Exclude obscure problems by min accepted count (default ≥1000) |
| **Solved Filter** | All / Exclude my solved / Only my solved (LeetCode session) |
| **Count** | Set number of problems to pick (1–20) |

### Codeforces / Programmers / SWEA

Each platform supports rating/level range + tag-based random picking with its own dialog.

## 한국어

LeetCode, Codeforces, 프로그래머스, SWEA에서 다양한 필터를 적용하여 랜덤으로 문제를 뽑을 수 있습니다.

### LeetCode

| 기능 | 설명 |
|------|------|
| **난이도 체크박스** | Easy, Medium, Hard 복수 선택 |
| **태그 칩** | Array, DP, Graph 등 태그를 칩으로 선택/제거 |
| **정답자 수 필터** | 정답자 N명 이상인 문제만 (기본 1000명) |
| **풀이 필터** | 전체 / 내가 푼 문제 제외 / 내가 푼 문제에서만 |
| **개수 설정** | 뽑을 문제 수 지정 (1~20개) |

### Codeforces / 프로그래머스 / SWEA

각 플랫폼별 레이팅/레벨 범위와 태그 기반 랜덤 뽑기를 별도 다이얼로그로 지원합니다.
