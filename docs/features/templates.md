## English

Save frequently used boilerplate as templates for quick access. Syntax-highlighted preview included.

### Features

- Save code snippets as named templates
- Organize by language
- Syntax-highlighted preview before inserting
- Quick insert into the current editor

## 한국어

자주 쓰는 코드를 템플릿으로 저장해두면 빠르게 불러올 수 있습니다.

### 기능

- 코드 스니펫을 이름 붙여 템플릿으로 저장
- 언어별 분류
- 삽입 전 구문 강조 미리보기
- 현재 에디터에 빠른 삽입
